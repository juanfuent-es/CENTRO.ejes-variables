La señalética urbana forma parte esencial de la comunicación visual en la ciudad. Sin embargo, muchas señales se producen mediante soluciones improvisadas que buscan adaptar el texto a distintos formatos, generando distorsiones tipográficas, problemas de legibilidad e inconsistencias visuales.

Esta documentación reúne ejemplos observados en el entorno urbano como punto de partida para analizar estas problemáticas y explorar cómo la tipografía variable puede utilizarse como herramienta para adaptar el texto sin deformar las letras, manteniendo claridad tipográfica, jerarquía visual y contraste adecuado.
