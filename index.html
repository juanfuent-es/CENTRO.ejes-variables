La señalética urbana suele resolverse mediante ajustes manuales que distorsionan la tipografía y afectan la legibilidad.

Mediante ejes tipográficos como peso, ancho o inclinación, este proyecto propone un sistema flexible para adaptar el texto a distintos formatos sin deformar las letras ni perder claridad, jerarquía visual y un contraste óptimo entre color y fondo.
